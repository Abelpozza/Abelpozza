# Login Screen App

Aplicação Android desenvolvida para estudo de arquitetura MVVM e Jetpack Compose.

## Tecnologias utilizadas

- Kotlin
- Jetpack Compose
- MVVM
- StateFlow

## Funcionalidades

- Tela de login
- Validação de campos
- Gerenciamento de estado
- Estrutura em camadas

## Arquitetura

O projeto segue arquitetura MVVM separando:

- data
- domain
- presentation
